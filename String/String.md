# 문자열
- 여러 개의 문자가 모인 집합체
```c#
string myString1 = "Hello";
char[] myString2 = {'H', 'e', 'l', 'l', 'o'};
```

## 문자열끼리 사칙연산(더하기)이 된다.

## 특수문자들 출력하는 방법이 따로 있다 ("\"", "\ㅜ",...)

## 문자열 포맷팅
```c#
string message1 = string.Format("Name: {0} / ID : {1}, name1, id1);
```

## 출력할때여러가지 서식들이 있다.

## $"..." 제일 좋음
-  string에 대입해도 된다
```c#
$"안녕하세요 {name} 학생! 학생번호 {id}입니다."
```
